# 領収書発行システム（再発行専用）

株式会社いえらぶパートナーズ向けの領収書再発行システムです。

## 🎯 システム概要

このシステムは、保証契約者（入居者）からの領収書発行依頼に対応するための社内向けWebアプリケーションです。

### 主な機能

- ✅ 5種類の保証料・手数料に対応
  - 初回保証料（非課税）
  - 月額保証料（非課税）
  - 更新保証料（非課税）
  - 集送金手数料（課税10%）
  - 請求事務手数料（課税10%）

- ✅ 2種類の発行形式
  - 通常の領収書（再発行）
  - 適格請求書（インボイス）

- ✅ 複数月対応
  - 月額保証料・集送金手数料・請求事務手数料は期間指定で明細自動生成

- ✅ 発行履歴管理
  - 直近10件の表示
  - CSV全件エクスポート

- ✅ PDF自動生成
  - 日本語完全対応
  - 再発行明記
  - 適格請求書発行事業者番号自動記載

## 📋 動作環境

- Python 3.8以上
- Flask 3.0.0
- ReportLab 4.0.7（PDF生成）
- SQLite 3（データベース）

## 🚀 セットアップ手順

### 1. 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 2. 日本語フォントのインストール（初回のみ）

```bash
sudo apt-get update
sudo apt-get install -y fonts-ipafont fonts-ipaexfont
```

### 3. アプリケーションの起動

```bash
python app.py
```

サーバーが `http://localhost:5000` で起動します。

## 💻 使い方

### 基本的な発行フロー

1. **契約者情報の入力**
   - 契約者名（必須）
   - 保証番号（任意・推奨）
   - 郵便番号（必須）
   - 住所（必須）
   - 物件名・号室（任意）

2. **領収内容の選択**
   - 発行する項目にチェック
   - 金額を直接入力（上下ボタンなし）
   - 複数月の場合は期間を指定

3. **発行設定**
   - 発行形式を選択（通常/インボイス）
   - 但し書きを選択（プリセットまたは自由入力）
   - 発行日を設定

4. **プレビュー確認**
   - 「プレビューを表示」ボタンをクリック
   - 内容を確認

5. **出力**
   - 印刷して郵送
   - PDFダウンロード
   - メール送信（開発中）

## ⚠️ 重要な注意事項

### 発行前の確認事項

1. ✅ 基幹システムで入金確認済みですか？
2. ✅ 契約書2枚目（お客様控え）の案内はしましたか？
3. ✅ 本当に再発行が必要ですか？

### 各項目の発行頻度と注意点

| 項目 | 頻度 | 注意点 |
|------|------|--------|
| 更新保証料 | 月10件程度 | 主に生活保護の方からの依頼 |
| 月額保証料 | 月数件 | 確定申告用など |
| 初回保証料 | ほぼゼロ | **原則発行しない**・契約書2枚目を案内 |
| 集送金手数料 | ほぼゼロ | インボイス対応必要 |
| 請求事務手数料 | ほぼゼロ | インボイス対応必要 |

### 課税・非課税の区分

**非課税項目（消費税なし）**
- 初回保証料
- 月額保証料
- 更新保証料

**課税項目（消費税10%）**
- 集送金手数料
- 請求事務手数料

### インボイス（適格請求書）について

**こんな時に使用:**
- 事業者（会社経営者・個人事業主）からの依頼
- 「会社の経費で落とす」と言われた場合
- 課税項目（集送金・請求事務手数料）の領収書
- 明確に「インボイスください」と言われた場合

**通常の領収書でOK:**
- 確定申告用（個人）
- 生活保護の提出用
- 特に指定がない個人のお客様

## 🆘 トラブルシューティング

### PDFが表示されない

→ ブラウザのポップアップブロックを解除してください

### 日本語が文字化けする

→ 日本語フォントがインストールされているか確認
```bash
ls -la /usr/share/fonts/opentype/ipafont-gothic/
```

### データベースエラー

→ `receipts.db` を削除してアプリを再起動
```bash
rm receipts.db
python app.py
```

### 入力した情報が消えた

→ ブラウザの戻るボタンは使わず、システム内のボタンで操作してください

## 📝 よくある質問（FAQ）

### Q1. 同じお客様から何度も依頼が来る場合は？

A1. 何度でも「再発行」可能です。すべて「再発行」表記が入りますので、法的問題はありません。

### Q2. 間違った内容で発行してしまった場合は？

A2. 正しい内容で再度発行してください。間違った領収書は破棄し、正しいものを送付します。

### Q3. 複数項目を一緒に発行できますか？

A3. できます。課税・非課税が混在していても自動的に区分して計算します。

### Q4. 発行履歴はいつまで残りますか？

A4. データベースに永続的に保存されます。CSV出力で月次報告などに活用できます。
